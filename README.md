# Charles and Patricia Heidelberger Foundation Website

A polished foundation website for the Charles and Patricia Heidelberger Foundation. The site presents tabbed sections for Home, Mission, History, Conferences, Board Members, and Contact. Originally made on Netlify, but edited on VSCode for Vercel, so this may not be accurate.

## Key Technologies

- TanStack Start
- React 19
- TanStack Router
- Tailwind CSS 4
- Netlify Forms
- Vite

## Local Development

Install dependencies, then run the development server:

```bash
npm install
npm run dev
```

The Netlify workflow validates production builds automatically after changes are submitted.

## Content Notes

Board member names, contact details, and conference specifics are currently written as editable placeholders. Update those sections in `src/routes/index.tsx` when approved foundation content is available.
